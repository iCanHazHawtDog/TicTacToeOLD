    System Requirements

    Hardware Requirements

    • x86 Processor or above
    • Minimum hardware required by the Java Virtual Machine
    • Network connection for Network based Multiplayer
    • A Monitor and Mouse

    Software Requirements

    • Any operating system that can run the Java Virtual Machine

    Main Menu

    This is the TicTacToe main application page.

    Access Help

    Help document can be accessed any time.

    Settings

    Settings provide the option of preference in terms of changing the game sound, player icon and maneuvering back to the main menu.

    Note: Changing sound and player icon functionality will be incorporated in Phase 2.

    High Scores

    This screen gives users the list of TicTacToe high scorers.

    Play

    Play button allows users to initialize the game. On its click, a second screen pops up which leads to initializing a two player game either in the same computer or on a network. To proceed, a player name is required after which either of the options: On this Machine or Network Based must be chosen. Port number 6262 needs to be opened in the local firewall.

    On this Machine

    Entering the player name and clicking on this menu allows you to enter two player's name and start playing the game.

    Network Based

    Clicking on this menu allows you to enter the person's name and act as a host machine or join a game.

    Host Machine

    Host machine allows you to host the game. If you choose this option, another player can connect to your computer and play on the game in host-guest architecture.

    Join Game

    This option will allow you to join the game hosted by others. While you need the host’s player IP address and the default static port number which is 6262, you also have to enter your name as the second player in order to play the network game.

    Play Game

    After the play board opens, the left top corner has information on the player names and their symbols (X or O), left bottom corner has information on player turn, and the right screen has the board. The players can play the game by clicking on the board cells during their turns only.

    Note: For more detailed help needed in learning how to play the game, click on the Wikipedia link, or watch the YouTube video.

    After each game round, a pop up screen shows up with the game details.

    Reset Button

    Reset button allows a player to reset the game board. All board cells are emptied.

    Exit

    When the exit button is clicked, the game session(s) is/are lost and the game is closed.
